Soluções HTML


Chama When
Código: 
<pega:choose>
	<pega:when test="NomeWhen">
	<!-- HTML or XML code to execute for first when -->
	</pega:when>
					
	<pega:when test="NomeWhen2">
	<!-- HTML or XML code to execute for second when -->
	</pega:when>
					
	<pega:otherwise>
    <!-- selected if no previous when tag evaluates to true -->
	</pega:otherwise>				
</pega:choose>

Chama Lista
Código: 

<pega:forEach name="Lista.pxResults">
<tr>
<td style='border-radius: 5px;border: 1px solid #0870b9; padding: 15px;text-align: left;vertical-align: middle;width: 60%;'> <pega:reference name="$THIS.Name" /> </td>
<td style='border-radius: 5px;border: 1px solid #0870b9; padding: 15px;text-align: left;vertical-align: middle;width: 20%;'> <pega:reference name="$THIS.Value" /> </td>
<td style='border-radius: 5px;border: 1px solid #0870b9; padding: 15px;text-align: left;vertical-align: middle;width: 20%;'>  <pega:reference name="$THIS.Height" /> </td>
</tr>
</pega:forEach>
