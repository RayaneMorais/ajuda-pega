Funções


replaceAll() usado para substituir todas as ocorrências da String selecionada por outra. O padrão pode ser uma string ou uma RegExp, e a substituição pode ser uma String ou uma função a ser chamada para cada ocorrência.
Exemplo: 
@String.replaceAll(Param.Teste,"ABC","DEF")

pxStringCSVFromPageList() usa dois parâmetros de entrada, uma propriedade Pagelist e uma propertyName, serve para transformar a lista em uma String.
Exemplo: 
@pxStringCSVFromPageList(Lista.pxResults,"STS_FILTRO")

IsInPageList() Retorna true, se o valor que foi passado como input estiver na propriedade que está na pagelist e false caso contrário.
Exemplo: 
@IsInPageList(.PropertyValue,"Value",Pagelist)

IsInPageListWhen() Retorne true, quando qualquer página tiver a correspondência.
Exemplo: 
@IsInPageListWhen(WhenName,Pagelist)

contains() usado para procurar um caractere ou uma sequência de caracteres. Retorna verdadeiro se encontrado e falso se não encontrou.
Exemplo: 
@contains(param.Teste,"abc")

containsIgnoreCase() usado para procurar um caractere ou uma sequência de caracteres, sem fazer distinção entre letras maiúsculas ou minúsculas.
Exemplo: 
@containsIgnoreCase(Param.Teste,"ABC")

equals() usado para compara duas expressões para determinar se elas são iguais e retorna verdadeiro ou falso.
Exemplo: 
@equals(param.Teste,"")

equalsIgnoreCase() usado para quando quer comparar uma String, sem fazer distinção entre letras maiúsculas ou minúsculas.
Exemplo: 
@equalsIgnoreCase(param.Teste,"aBc")

countInPageList() Retorna a contagem das páginas que têm a correspondência (propriedade).
Exemplo: 
@countInPageList("Value","PropertyName",Pagelist)

countInPageListWhen() Retorna a contagem das páginas que têm a correspondência (when).
Exemplo: 
@countInPageListWhen("WhenName",Pagelist)

IsInEachPageofList() Retorna verdadeiro, se em todas as páginas contiver o valor exatamente igual com a propriedade na Pagelist.
Exemplo: 
@IsInEachPageofList("PropriedadeOuValor","PropertydaPage",Pagelist)

IsInEachPageofListWhen() Retorne true, se a condição when for avaliada como para cada página da lista de páginas.
Exemplo: 
@IsInEachPageofListWhen("NomeWhen",Lista.pxResults)

SizeOfPropertyList() usado para verificar o tamanho da lista de valores. Retorna verdadeiro ou falso.
Exemplo: 
@SizeOfPropertyList(ListaAccessGroups.pxResults)>0

LengthOfPageList() usado para verificar o tamanho da page list.
Exemplo: 
@Utilities.LengthOfPageList(Page.Lista)

length() usado para verificar a quantidade de caracteres.
Exemplo: 
@String.length(Param.Teste)

IndexInPageList() Retorna o índice da primeira entrada na Pagelist, quando qualquer página tiver a correspondência (propriedade).
Exemplo: 
@IndexInPageList("ApplyJobs","pyLabel",D_ABC.pxResults)

IndexInPageListWhen() Retorna o índice da primeira entrada na Pagelist, quando qualquer página tiver a correspondência (when). Retorne -1 quando nenhuma correspondência for encontrada.
Exemplo: 
@IndexInPageListWhen("IsAvailable",D_ABC.pxResults)

entrySatisfiesCondition() Retorna verdadeiro se todos/algum dos valores atendem à comparação.
Exemplo: 
@entrySatisfiesCondition(Propriedade,"Comparador", Valor,"Opção")

pxReplaceAllViaRegex() usado para fazer pesquisa e substituição de correspondência de padrão no texto.
Exemplo: 
@String.pxReplaceAllViaRegex(param.Teste,"[A-z]","")

pzGenerateUniqueID() usado para gerar um código único com um prefixo inicial.
Exemplo: 
@pzGenerateUniqueID(tools,"PF")

getDataSystemSetting() usado para chamar/verificar DSS (Data System Setting).
Exemplo: 
@Utilities.getDataSystemSetting(Classe,NomeDSS)

pxExecuteAnActivity() usado para chamar Activity dentro do Data Transform.
Exemplo: 
@Utilities.pxExecuteAnActivity(Classe, NomeActivity)

DecisionTable.ObtainValue() usado para chamar Decision Table dentro do Data Transform.
Exemplo: 
@DecisionTable.ObtainValue(tools,Classe,"NomeDecisionTable",true)

CurrentDateTime() usado para gerar data/hora atual no formato GMT.
Exemplo: 
@(Pega-RULES:DateTime).CurrentDateTime()

endsWith() usado para verificar se corresponde aos últimos caracteres da String.
Exemplo: 
@endsWith(Param.Teste,":")

concat() usado para concatenar String.
Exemplo: 
@concat(Param.prefixo,Param.sufixo)

whatComesAfterLast() usado para retornar tudo o que vem depois do valor fornecido.
Exemplo: 
@(Pega-RULES:String).whatComesAfterLast(Param.Teste, ':')

whatComesBeforeFirst() usado para retornar tudo o que vem antes do valor fornecido.
Exemplo: 
@(Pega-RULES:String).whatComesBeforeFirst(Param.Teste, ':')

pxSortPageList() Retorna a ordem de classificação na mesma pagelist.
Exemplo: 
@pxSortPageList(Pagelist.pxResults,"Propriedade","Opção")

RemoveDeletedObjects() usado para excluir objetos dentro de um page list. Deve excluir todos que a propriedade .pyDeletedObject for setada com o valor "true".
Exemplo: 
@RemoveDeletedObjects(Primary.Lista)

pzDeleteAllInstancesOfDeclarativePage() usado para excluir todas as instâncias do data page.
Exemplo: 
@(Pega-RULES:DeclarePages).pzDeleteAllInstancesOfDeclarativePage
(tools, "D_Teste")

pxLogMessage() Registra a mensagem para o nível especificado nos logs do pega (d, i, f, w, e). Depois de executar a regra, pode ver o seguinte log no PegaRules.
Exemplo: 
@Default.pxLogMessage("Mensagem",'f')
